## Task-Focus Timer 

A high-utility productivity tool designed to help you focus on tasks using the Pomodoro technique. The timer switches between work and break cycles and plays a notification sound when the time is up. Its sleek, minimalist interface makes it easy to use and visually appealing.

## Features

Start, Pause, and Reset buttons

Customizable Work and Break durations

Automatic switch between work and break sessions

Notification sound when timer ends

Minimal glassmorphic design

Responsive for desktop and mobile devices

## Technologies Used

HTML – Structure of the app

CSS – Styling and glassmorphism effect

JavaScript – Countdown logic, session management, notifications

## How to Use

Enter desired Work and Break durations in minutes.

Click Start to begin the timer.

Timer automatically switches between work and break cycles.

When time is up, a notification sound will alert you.

Use Pause or Reset buttons to control the timer manually.
